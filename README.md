# Invexia

Este repositorio contiene el desarrollo de mi aplicación web realizada con **Django** para el modulo de desarrollo de aplicaciones web de segundo curso dentro del ciclo formativo de **Administración de Sistemas Informáticos en Red (ASIR)**.

## Motivos detras de la app

Durante mis prácticas de primer curso, me di cuenta de una necesidad importante que tenía la empresa donde estuve. El inventariado de los equipos se realizaba a papel, anotando manualmente la disponibilidad de cada equipo, quién poseia el equipo, la localizacion del equipo y la respectiva información de cada equipo. Aunque este método cumplía su función era poco práctico: resultaba complicado mantener los registros actualizados, había riesgo de pérdida de la informacion y las búsquedas eran lentas, lo que dificultaba tener un control goblal de los equipos. Con esta herramienta se podrán registrar de manera centralizada los dispositivos, sus caracteristicas y sus ubicaciones, ademas de poder reservar equipos disponibles.

## Descripción general

Mi objetivo con esta aplicacion es modernizar y agilizar el proceso, facilitando el trabajo de gestión reduciendo errores y ofreciendo una solución mas organizada y accesible que el inventario en papel, integrando y aplicando conocimientos adquiridos en clase, incluyendo:

- Desarrollo Web con Django (Python).
- Gestión de bases de datos.
- Despliegue de aplicaciones en entornos reales o virtualizados.
- Control de versiones con Git y GitHub
  
## La Empresa 
La empresa consta de un almacen con 20 portatiles actualmente. Los portatiles llevan instalados dualboot y otros single boot, variando entre sistemas que usan distros de linux como Ubuntu, Ubuntu server, Arch y Mint, tambien versiones de windows que oscilan entre windows 7, 10 y 11. En la empresa suelen llegar portatiles nuevos y tambien se desechan otros en puntos limpios de la ciudad. En cuanto al edificio de la empresa, hay un total de 3 habitaciones (T1, T2 y T3), más el almacén donde se almacenan los portatiles.
